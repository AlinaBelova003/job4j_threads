# job4j_threads
## Проект для отработки навыка работы с git.
### В проекте выполняется работа:
* Работа с ветками
* Добавление изменений в последний commit
* Откат изменений
* Объединение нескольких commit
* Временное хранение
* Слияние веток
* Конфликты слияния
* Удаление ветки и создание ветки из commit
